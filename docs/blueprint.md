# **App Name**: PersonaLink

## Core Features:

- Profile Display: Showcase a profile photo and brief bio.
- Skills Section: Display a list of skills with proficiency levels.
- Projects Showcase: Display a portfolio of projects with descriptions and links.
- Interactive Contact Form: Enable users to send messages directly.
- AI-Powered Bio Tool: Generate alternative bios based on listed experience and goals, using a tool to highlight certain elements or suggest variations in tone.

## Style Guidelines:

- Primary color: Teal (#008080) to give a modern and clean feel.
- Background color: White (#FFFFFF) for a minimalist look.
- Accent color: Light Green (#90EE90) to draw attention to calls to action.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, readable design. Note: currently only Google Fonts are supported.
- Profile photo prominently displayed on the right, with content sections arranged linearly on the left.
- Simple, professional icons for skills and contact information.
- Subtle fade-in effects when sections load, and smooth scrolling for navigation.
